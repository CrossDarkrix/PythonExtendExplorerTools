# Python Extend Explorer Tools

マルチプラットフォームのファイラーとファイル検索ツール

# ビルドに必要なもの

・PySide6(GUIを提供してくれるやつ)

・Py7zr(7Zip形式のファイル圧縮をPythonで扱うモジュール)

・send2trash(削除したファイルをゴミ箱に入れてくれるモジュール)

・psutil(コマンドラインツールのモジュール)

・pyqtgraph(グラフ作成モジュール)

・numpy
